如果我们令f[i] = a[i] - a[i-1] f[0] = a[0] ,f[n] = -a[n-1] 
f 为  a 的差分数组
可以发现，差分数组具有以下性质：
1.原数组中所有元素都等于零，等价于差分数组中所有元素都等于零。
2.如果我们将原数组中以a[i] 为 开头且长度为k的子数组每个元素都减去1，
  等价于 f[i] -1,,f[i+k] + 1
  比如 原数组 2 2 3 1 1 0
           f 2 0 1 -2 0 -1 0
3.考虑 f[0] :
  如果 f[0] < 0 ：则无解
  如果 f[0] > 0: 需要对f[0] 进行-1 操作 直至 f[0] = 0
                 等价于 对 f[k] 进行 f[0] 次 操作。
